-------------------------------------------------


CardWirth用　ライフゲージ素材


-------------------------------------------------

「カードワース」用差し替えライフゲージ素材の詰め合わせです。
実用性というより雰囲気重視のファンシー系が主となっております。
お好きに組み合わせたり加工したりして楽しく遊んでいただけると嬉しいです。


◆使用方法◆　------------------------------------

使いたい画像を
「STATUS_LIFEGUAGE.bmp」「STATUS_LIFEBAR.bmp」
にリネームし、Resourceフォルダに入れてください。

（Pyの場合はフォルダ・ファイル名が異なるようです。
「LIFEGUAGE.bmp」「LIFEBAR.bmp」
にリネームし、使用中のスキンフォルダの
Resource\Image\Statusに入れてください）


◆せつめい◆　------------------------------------

・STATUS_LIFEGUAGE_01.bmp（四角）
そこそこ見やすく、どのライフバーとも合わせやすいかと思います。

・STATUS_LIFEGUAGE_02.bmp（星形）
視認性は落ちますが、ランダムな色合いのものと合わせると
金平糖ちっくでいいと思います。

・STATUS_LIFEGUAGE_03.bmp（星形２）
02の派生です。多少見やすいかと思います。

・STATUS_LIFEGUAGE_04.bmp（ふつうの）
自分が普段使ってるのも入れときます。
デフォルトに近い形です。ちょっと細身。
目が疲れたらこちらをどうぞ。
